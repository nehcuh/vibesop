### 3.2 AI 路由架构

**5 层路由系统**:
```
Layer 0: AI Semantic Triage (95% 准确率)
  ├─ Multi-Provider (Claude/OpenAI)
  ├─ Multi-Level Cache (70%+ 命中)
  ├─ Context-Aware Matching
  └─ ~50-150ms 延迟

Layer 1: Explicit Overrides (100% 准确率)
  └─ 用户指定技能

Layer 2: Scenario Patterns (85% 准确率)
  └─ 预定义场景匹配

Layer 3: Semantic Matching (75% 准确率)
  └─ TF-IDF + 余弦相似度

Layer 4: Fuzzy Matching (60% 准确率)
  └─ Levenshtein 距离
```

**环境智能检测**:
```ruby
# Claude Code 内部自动禁用外部 API
if running_in_claude_code?
  @enabled = false unless ENV['VIBE_AI_TRIAGE_ENABLED']
end
```

**评估**: ✅ **架构优秀，实现完整**
