## External Content Processing — Detailed Template

**Value extraction dimensions (lightweight)**:
- Learnable: methods/techniques/tools/configs
- Actionable: things to do right now
- Thought-provoking: paradigm-shifting points

**Interaction type decision**:
| Scenario | Suggestion |
|----------|-----------|
| Hot topic + have perspective | Quote tweet |
| Influencer + can add value | Comment (gain exposure) |
| Good content + nothing new to add | Simple repost |
| Doesn't match positioning | Skip interaction |

**Upgrade to full analysis**:
- User says "detailed analysis" / "dig deeper" / "anything else" -> run 13-dimension analysis

## Experience Deposit Capture

**Signals (three types)**:

1. **Pitfall record**: Stuck 15+ minutes, multi-round debugging, counter-intuitive behavior
2. **Decision pattern**: Non-obvious technical choice, trade-off
3. **Cognitive upgrade**: Changed thinking, deeper understanding, reusable model

**Action**:
```
**Experience deposit**: [one-line summary] ([type])
Record to patterns.md?
```
